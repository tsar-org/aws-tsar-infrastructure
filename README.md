# aws-tsar-infrastructure

## Requirement

- docker(<https://www.docker.com/ja-jp/>)
- task(<https://taskfile.dev/installation/>)
- lefthook(<https://lefthook.dev/installation/index.html>)

## How to Develop

1. `.env`に必要な情報を記述

    ```text
    AWS_ACCESS_KEY_ID=""
    AWS_SECRET_ACCESS_KEY=""
    AWS_SESSION_TOKEN=""
    ```

1. 環境立ち上げ

    ```sh
    task up
    ```
